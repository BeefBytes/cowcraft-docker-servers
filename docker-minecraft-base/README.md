# docker-minecraft-base
Base image for various BeefBytes minecraft projects

| MAX_PLAYERS      | 100            |
| PORT             | 25565          |
| VIEW_DISTANCE    | 10             |
| WHITELIST        | false          |
| GAMEMODE         | survival       |
